# Luvly Lounge

A modern, safe dating app for women.

## Features
- **Smart Matching**: Find partners based on shared interests.
- **Safe Space**: Strict verification (camera gestures) and onboarding.
- **Real-Time Chat**: Instant messaging with auto-delete for safety.
